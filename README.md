# 🎓 منصة فهمي التعليمية (Fahmni LMS)

منصة **"فهمي"** هي نظام متطور ومتكامل لإدارة التعلم (Learning Management System - LMS) مصمم خصيصاً للمدرسين والطلاب في العالم العربي. توفر المنصة تجربة تعليمية حديثة، تفاعلية، ومحمية بالكامل مع واجهات مستخدم متميزة وتأثيرات بصرية راقية.

---

## 🛠️ بنية النظام والتقنيات المستخدمة (Tech Stack)

تم بناء المنصة باستخدام أحدث التقنيات لضمان السرعة، الأمان، وقابلية التوسع:

*   **الواجهة الأمامية (Frontend):**
    *   **React 19** & **TypeScript** لضمان كود منظم وآمن الأنواع.
    *   **Vite** كأداة بناء وتطوير فائقة السرعة.
    *   **Tailwind CSS v4** لتصميم واجهات عصرية ومتجاوبة بالكامل مع دعم الوضع الداكن (Dark Mode).
    *   **Framer Motion** لإضافة انتقالات وتأثيرات حركية سلسة وجذابة.
    *   **Recharts** لبناء لوحات إحصائية ورسوم بيانية تفاعلية للمدرسين والإدارة.
*   **الخلفية (Backend):**
    *   **Node.js** & **Express** مع **TSX** لتشغيل سيرفر التطوير الخاص بالخدمات المتقدمة.
    *   **Puppeteer** لإجراء عمليات أتمتة وتوليد ملفات تفاعلية.
*   **قاعدة البيانات والخدمات السحابية (Database & Cloud Services):**
    *   **Firebase Authentication** لإدارة تسجيل الدخول الآمن للمستخدمين.
    *   **Firestore Database** لتخزين البيانات بشكل لحظي وسريع (Realtime).
    *   **Firebase Storage** لرفع واستضافة المستندات والواجبات وصور الحسابات.
    *   **Cloudflare Workers** لتسريع الاستجابة وتحسين البنية التحتية.
*   **الذكاء الاصطناعي والدفع الإلكتروني:**
    *   دمج **Gemini AI API** لتقديم تحليلات ذكية ومساعدة الطلاب.
    *   بوابة دفع **Stripe** لتمكين عمليات الدفع الإلكتروني المباشر وشحن المحفظة.

## 👥 مميزات المنصة الرئيسية حسب نوع المستخدم

### 👨‍🎓 بوابة الطالب (Student Portal)
*   **لوحة تحكم ذكية:** لمتابعة الكورسات المشتركة، التقدم الدراسي، والشهادات المحصّلة.
*   **مشغل فيديو ذكي ومحمي:** يمنع تحميل الفيديوهات أو تصفح المحتوى بشكل غير مصرح به، مع دعم استئناف التشغيل.
*   **نظام المحفظة الإلكترونية (Wallet):** شحن الرصيد من خلال أكواد الشحن (Coupons/Cards) أو الدفع الإلكتروني.
*   **الاختبارات والواجبات التفاعلية:** إمكانية حل الكويزات وتلقي التصحيح والتعليقات بشكل فوري.
*   **بنك الأسئلة والمكتبة:** تصفح وتحميل المستندات والمراجعات التعليمية.

### 👨‍🏫 بوابة المعلم (Teacher Portal)
*   **إدارة الكورسات:** إضافة وتعديل الكورسات، الدروس، والمحاضرات بسهولة.
*   **منشئ الاختبارات المتقدم:** وضع أسئلة متعددة الاختيارات أو مقالية وتحديد الدرجات.
*   **نظام المناقشات:** التفاعل الإيجابي مع استفسارات الطلاب داخل كل درس.
*   **تقارير الأداء:** إحصائيات تفصيلية لدرجات الطلاب ونسبة حضورهم وتفاعلهم.

### 👑 لوحة تحكم الأدمن والمدراء (Admin & Super Admin Dashboard)
*   **إعدادات الموقع (Site Settings):** تخصيص الهوية البصرية (اللوجو، ألوان المنصة، العنوان) مباشرة من لوحة التحكم.
*   **توليد أكواد الشحن:** إنشاء وإدارة كوبونات شحن المحفظة وحذفها تلقائياً بمجرد الاستخدام لمنع التلاعب.
*   **قارئ كود الـ QR للغائبين والحضور:** تسجيل حضور الطلاب للمحاضرات الفعلية عبر مسح كود QR مخصص.
*   **الإدارة المالية:** متابعة الإيرادات، طلبات شحن المحافظ، والعمليات المالية مع إحصائيات بصرية.

## 🚀 دليل التشغيل والتهيئة السريعة

### المتطلبات الأساسية
1. تثبيت بيئة تشغيل **Node.js** (الإصدار 18 أو أحدث).
2. تثبيت مدير الحزم **NPM** (يأتي مدمجاً مع Node.js).

### خطوات التثبيت والتشغيل المحلي
1. قم بتهيئة ملف المتغيرات البيئية من خلال نسخ `.env.example` إلى ملف جديد باسم `.env`:
   ```bash
   cp .env.example .env
   ```
2. قم بتثبيت المكتبات والاعتمادات المطلوبة:
   ```bash
   npm install
   ```
3. ابدأ تشغيل المنصة في بيئة التطوير المحلية:
   ```bash
   npm run dev
   ```
4. لبناء النسخة المخصصة للإنتاج (Production Build):
   ```bash
   npm run build
   ```

## ⚙️ تهيئة ربط الـ Backend (Firebase & APIs)

تعتمد المنصة بشكل أساسي على **Firebase** لإدارة قواعد البيانات والمصادقة. لتشغيل مشروعك الخاص:

1. توجه إلى [كونسول فايربيز (Firebase Console)](https://console.firebase.google.com/) وأنشئ مشروعاً جديداً.
2. قم بتفعيل الخدمات التالية:
   *   **Authentication:** فعل خيار الدخول بالبريد الإلكتروني والكلمة السرية.
   *   **Firestore Database:** أنشئ قاعدة البيانات في وضع native.
   *   **Storage:** أنشئ حاوية تخزين لرفع الملفات والواجبات.
3. انسخ إعدادات الاتصال (Config Object) من صفحة المشروع وضعها في الملف التالي داخل الكود:
   *   [src/lib/firebase.ts](file:///d:/%D9%82%D8%B1%D8%A7%D9%86/fahmni/src/lib/firebase.ts)
4. قم بنشر قواعد الحماية لقواعد البيانات والتخزين باستخدام الملفات المتوفرة بالمشروع:
   *   قواعد Firestore في [firestore.rules](file:///d:/%D9%82%D8%B1%D8%A7%D9%86/fahmni/firestore.rules)
   *   قواعد التخزين في [storage.rules](file:///d:/%D9%82%D8%B1%D8%A7%D9%86/fahmni/storage.rules)
