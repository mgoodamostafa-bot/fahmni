# 🎓 منصة فهمي (Fahmni LMS) - النسخة النهائية المطورة

مرحباً بك في منصة "فهمي" التعليمية، النظام المتكامل لإدارة التعلم بأسلوب عصري ومحمي.

---

## 🚀 دليل التشغيل السريع

### المتطلبات الأساسية
1. **Node.js**: إصدار 18 أو أحدث.
2. **NPM**: يأتي مع Node.js.
3. **Firebase Project**: حساب على Firebase لإدارة البيانات.

### التثبيت والتشغيل
1. قم بفك ضغط المشروع في مجلد محلي.
2. افتح مبدل الأوامر (Terminal) في مجلد المشروع.
3. قم بتثبيت التبعيات:
   ```bash
   npm install
   ```
4. ابدأ تشغيل المنصة في وضع التطوير:
   ```bash
   npm run dev
   ```

---

## ⚙️ ربط Firebase (Backend)

المنصة تعمل كلياً عبر متصفح العميل (Client-side) مع الربط المباشر بـ Firebase.

1. اذهب إلى [Firebase Console](https://console.firebase.google.com/).
2. أنشئ مشروعاً جديداً.
3. أضف تطبيق Web واحصل على كود الربط (Config Object).
4. افتح ملف `src/lib/firebase.ts` وقم بتحديث الثوابت ببيانات مشروعك.
5. قم بتفعيل الخدمات التالية في كونسول Firebase:
   - **Authentication**: تفعيل الدخول بالبريد الإلكتروني (Email/Password).
   - **Firestore Database**: إنشاء قاعدة البيانات في وضع الـ Native.
   - **Storage**: لرفع ملفات الواجبات وصور المدرسين.

## 🎨 التخصيص (Branding & Colors)

### تغيير اللوجو (Logo)
- يمكنك تغيير اللوجو الخاص بالمنصة من خلال **لوحة تحكم الأدمن (Admin Dashboard)** -> **إعدادات الموقع (Site Settings)** ورفع الصورة الجديدة. سيتم تحديثها فوراً في كل الصفحات.

### تغيير الألوان (Theme)
- المنصة تعتمد على **Tailwind CSS v4**.
- لتعديل الألوان الأساسية، افتح ملف `src/index.css`.
- الوضع الداكن (Dark Mode) مضبوط الآن على **الكحلي الملكي (#0a192f)**.
- الوضع الفاتح (Light Mode) مضبوط على **الأبيض الصريح (#ffffff)**.

---

## 👥 مميزات المنصة (Key Features)

- **سايدبار ذكي (Smart Sidebar)**: ثابت على اليمين في اللاب توب ومساحة مريحة للمحتوى، وDrawer شفاف في الموبايل.
- **نظام المحفظة (Wallet System)**: شحن الرصيد عبر الأكواد (Coupons) المدمجة مع حذف الكود آلياً فور الاستخدام.
- **مشغل فيديو احترافي**: حماية المحتوى، دعم Fullscreen كامل، وسحب تلقائي لصور اليوتيوب بجودة عالية.
- **دخول آمن**: المنصة مغلقة بالكامل (Full Auth) ولا يمكن تصفح المحتوى إلا للمستخدمين المسجلين.
